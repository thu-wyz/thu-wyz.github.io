# Assets

Drop your images here. The page references these filenames:

- `photo.jpg`   — your profile photo (square works best, e.g. 400×400)
- `paper1.png`  — thumbnail/figure for the first selected work
- `paper2.png`  — thumbnail/figure for the second selected work

Until these files exist, the page shows gray placeholders automatically
(via the `onerror` fallback in `index.html`), so the site never looks broken.
